Documentation

Usage Guide

  1. Open the Home page.
  2. Select a history period (e.g., 1y for one year of data).
  3. Enter the forecast horizon (e.g., 7 days).
  4. Click Predict Now to generate results.
  5. View visualized data and predicted future prices.

Backend Overview

The system retrieves stock data using yfinance or a dataset, scales values with MinMaxScaler, trains an LSTM model in TensorFlow, and predicts the next N closing prices.

Frontend Features

  • Dynamic charts using Chart.js
  • Clean responsive design with CSS3
  • Instant feedback with AJAX-based prediction

Model Architecture

The deep learning model uses:

  • 2 LSTM layers (64 units each)
  • Dropout regularization
  • Dense output layer (1 neuron for closing price)
